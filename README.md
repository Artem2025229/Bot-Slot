Привет!🖐️
Этот проект - заготовка для телеграм-бота со слотами🎰. Бот содержит в себе основные функции, необходимые для работы. при желании вы можете добавить новые хендлеры или видоизменить старые.
---


📚 Для начала использования бота необходимо установить библеотеку telebot, для установки нужно вписать в терминале команду `pip install pyTelegramBotAPI`, потом в файле __config__ заменить в скобках слово *TOKEN* на свой токен, который можно получить у [BotFather](https://telegram.me/BotFather). На этом всё, ваш бот полностью готов к использованию!
